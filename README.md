# E-Shop
Open two terminals: 
  Terminal -1:
    -> npm install
    -> cd E-Shop/client/
    -> npm run dev
  Terminal -2: 
    -> npm install
    -> cd E-Shop/server/
    -> create .env file and ADD these lines
      -> API_URL = /api/v1
      -> MONGO_CONNECTION_STRING = {Add the MongoCollection link}
      -> secret = {Note: Keep any random value Ex:sdfngf@g1742He}
    -> npm start
